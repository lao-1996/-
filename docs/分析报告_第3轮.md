# 第3轮分析报告

## 美化待办

- B16 **粒子系统增强**：当前子弹命中效果仅用简单矩形表示，缺乏冲击感。建议在 `drawExplosion()` 函数中添加粒子系统，发射多方向扩散的圆形/星形粒子，并加入 alpha 渐变消失动画，提升爆炸视觉层次。

- B17 **玩家飞机拖尾特效**：玩家飞机（`drawPlayer()`）移动时缺少引擎喷焰或运动轨迹拖尾效果。建议在飞机尾部绘制动态火焰粒子或发光线条（使用 `shadowBlur` + `createRadialGradient`），增强速度感与科技感。

- B18 **关卡切换过渡动画**：波次（Wave）切换时直接刷新，体验生硬。建议增加全屏扫光或文字"Wave X"放大淡出动画，使用 Canvas `globalAlpha` 渐变配合计时器实现，持续约 1.5 秒。

- B19 **UI 面板玻璃化与发光边框**：当前得分、生命值等 HUD 使用纯文字绘制，视觉层次单薄。建议为 HUD 区域绘制半透明圆角背景框（`rgba` + `roundRect`），并添加青色/紫色发光描边（`shadowColor` + `shadowBlur`），打造赛博朋克风格 UI。

- B20 **敌人死亡闪白效果**：敌机被击中死亡时无视觉反馈，仅消失。建议在敌机被击杀的帧中，先以纯白色绘制同形状覆盖层（`globalCompositeOperation: 'source-over'`），然后1帧后恢复正常消失，产生"闪白"击杀反馈。

- B21 **星空背景动态层次化**：当前背景星点为单一速度滚动，缺乏纵深感。建议将星星分为 3 层（近/中/远），速度分别为 4x/2x/1x，大小分别为 2.5/1.5/0.8px，形成视差滚动（Parallax）效果，增强飞行沉浸感。

- B22 **技能/道具拾取光效**：道具出现时无吸引注意的动态效果。建议为道具物品添加旋转光晕动画（用 `rotate()` + 渐变圆环每帧旋转5°），并在拾取瞬间播放圆形扩散波纹动画。

- B23 **Boss 血条美化**：Boss 血条为简单矩形条，缺乏辨识度。建议将其替换为分段式血条（每段代表25%血量），加入渐变色（满血绿→半血黄→低血红），并在每段边界绘制分隔线，血条背景添加发光底框。

## 功能待办

- F13 **本地存档/读档系统**：游戏无任何进度保存机制，刷新即丢失。建议使用 `localStorage` 保存最高分（`highScore`）、已解锁的飞机皮肤（如有）和当前解锁波次进度，在主菜单显示历史最高分，并在 Game Over 界面对比当前得分与记录。

- F14 **难度选择模式**：当前游戏只有单一难度曲线，无法照顾不同水平玩家。建议在主菜单增加"简单/普通/困难"选项，分别对应：敌机血量 ×0.7/×1/×1.4，生成速度 ×0.8/×1/×1.2，子弹速度 ×0.8/×1/×1.3，同时在困难模式 Boss 增加新攻击模式。

- F15 **暂停菜单完善**：当前 ESC/P 键暂停后界面简陋，缺少选项。建议在暂停界面增加"继续游戏""重新开始""返回主菜单""音量调节"四个可交互按钮，使用 Canvas 绘制半透明遮罩 + 菜单项，通过 `mousemove` 实现悬停高亮。

- F16 **连击系统（Combo）**：当前消灭敌人仅获得固定分数，缺乏节奏奖励。建议实现连续击杀计数器：在 2 秒内连续击杀时计数累加，连击数 ×10% 加成分数（最高 500% 上限），屏幕中央浮现"×3 COMBO!"等文字提示，超过5连击触发短暂全屏紫光闪烁。

- F17 **碰撞检测边缘 Bug**：当前碰撞检测使用矩形 AABB，对于旋转的敌机或非规则子弹存在明显空隙误差（子弹飞过但未命中，或远未碰触即爆炸）。建议对玩家和 Boss 改用缩小 20% 的碰撞盒（hitbox shrink），对普通敌机使用圆形碰撞检测（`Math.hypot(dx,dy) < r1+r2`）以提高准确性。

- F18 **音效系统**：游戏完全无音频反馈，沉浸感大幅缺失。建议使用 Web Audio API（`AudioContext`）生成程序化音效：射击音（短促方波脉冲）、爆炸音（白噪声衰减）、拾取道具音（上行音调）、Boss 出场音（低频震动），全部无需外部文件，纯 JS 合成。

- F19 **新手引导/教程**：首次进入游戏缺少任何操作说明。建议在第1波开始前播放3秒高亮动画：箭头指向WASD/方向键区域说明移动，闪烁提示空格/Z键说明射击，自动消失后开始游戏，同时在第1波第1个敌机出现时显示"击败敌机！"提示文字。

- F20 **波次间休息/补给阶段**：当前波次切换无间隔，玩家无法喘息。建议在每波结束与下一波开始之间增加5秒"补给阶段"：血量恢复10%、屏幕上方掉落随机道具×2、倒计时条可视化显示，营造战略节奏感。

- F21 **成就系统**：游戏缺乏长期游玩动力。建议实现5个基础成就：首次击杀Boss（"猎手"）、无伤通关第3波（"无懈可击"）、单波击杀50敌机（"机海战术"）、使用道具5次（"装备控"）、总游戏时间超过10分钟（"老兵"），成就解锁时右下角弹出动画通知条。

## 优先级排序

1. **F17 碰撞检测边缘 Bug**（功能）：这是直接影响游戏公平性和手感的核心问题，玩家会明确感知到"明明没碰到却死了"或"明明打中了没扣血"，优先修复可立即提升游戏可信度。

2. **F18 音效系统**（功能）：完全无声的游戏体验缺失感强烈，使用 Web Audio API 无需任何外部资源即可实现，投入产出比极高，能大幅提升沉浸感和打击感。

3. **B21 星空背景视差滚动**（美化）：改动成本低（仅需将现有星点数组分组），视觉提升效果显著，能立即让游戏场景从"扁平"变成"有纵深"，是性价比最高的美化项。
