# 🎮 奔赴心意的冒险 - 完整增强版

## 📖 项目简介

这是一个HTML5 Canvas平台的横版闯关游戏，讲述了一个浪漫的冒险故事。玩家需要帮助主角"捞"穿越重重阻碍，收集星光碎片，最终奔赴心爱之人"小秋"的身边。

## 🚀 快速开始

### 1. 启动游戏
```bash
# 在项目目录启动HTTP服务器
python -m http.server 8000

# 然后在浏览器访问
http://localhost:8000
```

### 2. 控制方式
- **键盘**：
  - ← → 左右移动
  - ↑ 跳跃（双击大跳）
  - ↓ 下蹲躲避
  - 空格 跳过对话

- **触屏**（移动设备）：
  - 屏幕按钮控制
  - 左侧：移动和跳跃
  - 右侧：下蹲和动作

### 3. 游戏目标
1. 收集关卡中的星光碎片（⭐）
2. 躲避或越过各种障碍
3. 到达关卡终点
4. 解锁所有关卡和隐藏结局
